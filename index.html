Lessons gained from Project
:
Gumawa sa separate files, then pagsama-samahin sa huli (merge) (para hindi malito)
Wag i-number ang sections para madali lang pagpali-palitin.
Magplan ng naming conventions sa unang una para hindi magulo sa pag name ng selectors
I-minimize kung maari ang CSS at ilagay sa attributes
Manage your time well, magplano para alam ang mga susunod na gagawin
Unahin ang madali, iwan ang mahirap (para sa huli, madali matapos yung mga madadali)

GIT - Open Source Distributed Version Control System

Clone a repo_url to a local folder

Fetch a repository/branch
git add <files>
Add changes to the staging area
git commit -m 'message'
Save changes to the repository
git push <repo_url>

git pull <repo_url>
Clone a repository into a blank folder
git status
Displays the state of the repository
git branch
Show/List existing branches
git branch <name>
Create a new branch
git checkout <branch>
Switch to the branch
git checkout -b <branch>
Create and Switch to the new branch
git branch -d <name>
Delete branch
git branch -m <new name>
Rename current branch to new name
git merge <branch>
Merge/Integrate branch to current

CombinatorsNext

Descendant

    property: "value";

Child

    property: "value";

}

ALL p elements immediately following a div as a sibling.

General Sibling

    property: "value";
}

ALL p elements following (not necessarily immediately after) a div as a sibling.

AttributeNext

    property: "value";
}

Select elements having the given attribute.

[attr="value"]

[attribute = "value"] {
    property: "value";
}

Select elements having the exact value for a given attribute.

[attr |= "value"]

[attribute |= "value"] {
    property: "value";
}

Select elements starting with the given word (value) for a given attribute.

[attr ^= "value"]

[attribute ^= "value"] {
    property: "value";
}

Select elements starting with the given value for a given attribute.

[attr $= "value"]

[attribute $= "value"] {
    property: "value";
}

Select elements ending with the given value for a given attribute.

[attr ~= "value"]

[attribute ~= "value"] {
    property: "value";
}

Select elements containing the word (value) for a given attribute.

[attr *= "value"]

[attribute *= "value"] {
    property: "value";
}

Select elements containing the given value for a given attribute.
